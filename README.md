🚀 INFNOVA Frontend Internship Engineering Challenge

A pixel-perfect frontend implementation based on the provided Figma design for the INFNOVA Academy internship process.

🌐 Live Demo

👉 https://infnova-frontend-internship-engineering-challenge-ne10jd87s.vercel.app/

🛠 Tech Stack

Next.js 14 (App Router)

TypeScript

Tailwind CSS

Deployed on Vercel

✨ Features

Clean, reusable component architecture

Fully responsive (mobile, tablet, desktop)

Pixel-accurate UI implementation from Figma

API integration (/api/courses, /api/courses/:id)

Loading states & graceful error handling

📦 Getting Started
git clone <your-repo-link>
cd <project-folder>
npm install
npm run dev

Open http://localhost:3000

Focused on clean UI structure, maintainable components, and production-ready patterns.
